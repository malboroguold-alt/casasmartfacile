# ✅ Checklist SEO - Nuovo Articolo

## Prima di scrivere
- [ ] Keyword principale identificata (es: "migliori lampadine smart 2026")
- [ ] Keyword secondarie trovate (3-5 variazioni)
- [ ] Ricerca competitor fatta (primi 3 risultati Google)
- [ ] Intento di ricerca chiaro (informazionale/commerciale)

## Struttura articolo
- [ ] Title tag: 50-60 caratteri, keyword all'inizio
- [ ] Meta description: 150-160 caratteri, call to action
- [ ] URL: breve, con keyword, trattini (es: /migliori-lampadine-smart-2026)
- [ ] H1: un solo H1 per pagina, con keyword
- [ ] H2: ogni 300-400 parole
- [ ] H3: per sotto-sezioni

## Contenuto
- [ ] Lunghezza: minimo 1500 parole (ideale 2000+)
- [ ] Introduzione: hook forte, problema/soluzione
- [ ] Paragrafi: max 3-4 righe
- [ ] Bullet list: almeno 3 per articolo
- [ ] Immagini: minimo 2, con alt text descrittivo
- [ ] Link interni: 3-5 verso altri articoli del sito
- [ ] Link esterni: 2-3 verso fonti autorevoli

## Link affiliati Amazon
- [ ] ASIN verificato (prodotto disponibile)
- [ ] Prezzo aggiornato
- [ ] Link con tag `casasmartfaci-21`
- [ ] Attributi: `rel="noopener noreferrer nofollow sponsored"`
- [ ] Target: `_blank` per aprire in nuova scheda
- [ ] Disclaimer affiliazione presente

## Ottimizzazione on-page
- [ ] Schema.org Article JSON-LD
- [ ] Open Graph tags (Facebook)
- [ ] Twitter Card tags
- [ ] Canonical URL corretto
- [ ] Breadcrumb navigation
- [ ] Tabella comparativa se articolo prodotto

## Dopo pubblicazione
- [ ] Sitemap.xml aggiornata
- [ ] URL sottomesso su Google Search Console
- [ ] Post social programmati (Instagram, Twitter, Reddit)
- [ ] Link interni da altri articoli esistenti
- [ ] Monitoraggio posizionamento (dopo 2-4 settimane)

## Metriche da tracciare
- [ ] Posizione keyword target
- [ ] Click da Google
- [ ] Tempo sul sito
- [ ] Bounce rate
- [ ] Click link Amazon
